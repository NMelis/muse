# Muse

Turn any folder with mp3s into a streaming music player.

## Installation & Tool Usage

```bash
# Install the latest tool.
go install github.com/prophittcorey/muse/cmd/muse@latest

# Serve your music.
muse --dir "~/Music" --host "0.0.0.0" --port "3000"
```
